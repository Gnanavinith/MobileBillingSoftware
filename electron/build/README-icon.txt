Place your branded Windows icon file at:

  MobileBillingSoftware/electron/build/icon.ico

Requirements:
- Format: .ico (multi-size 16, 32, 48, 64, 128, 256px recommended)
- Color depth: 32-bit with alpha

This icon will be used for:
- App executable icon
- Installer icon
- Desktop and Start Menu shortcuts

After adding the icon, run from electron folder:

  npm run dist
